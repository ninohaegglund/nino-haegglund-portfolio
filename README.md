# Nino Hägglund Portfolio

A personal portfolio and online CV for showcasing my work as a junior .NET developer. The site highlights selected projects, technical skills, experience, contact details, and a downloadable CV.

## About the Portfolio

This portfolio is built to give recruiters, employers, and collaborators a quick overview of who I am, what I build, and the technologies I work with. The main focus is on backend and full-stack projects using C#, ASP.NET Core, React, and cloud-related tools.

## What It Includes

- Featured project carousel with screenshots, tech stacks, GitHub links, live demos, and case study details
- Skills grouped by backend development, cloud and DevOps, frontend, tools, architecture, and authentication
- Experience timeline with education and internship work
- Contact section with email, location, GitHub, and LinkedIn links
- Light and dark theme support
- Downloadable CV from the public assets folder

## Featured Projects

- **Personal Job Application Manager** - Full-stack job search and application tracking app with authentication.
- **E-Commerce Microservices Platform** - E-commerce system built with multiple ASP.NET Core services and a React frontend.
- **PC Builder** - ASP.NET Core MVC and Web API application for custom PC orders.
- **Project Manager MVC** - Project management dashboard prototype with full-stack features.
- **Alien Vision Prototype** - Unity 2D game prototype focused on a reveal/hide vision mechanic.
- **Dr. News CMS** - Multilingual Umbraco CMS news site with accessibility-focused implementation.

## Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS
- **UI:** shadcn/ui, Radix UI, Lucide React
- **State and utilities:** React Query, React Router, custom hooks
- **Testing and quality:** Vitest, Testing Library, ESLint
- **Portfolio focus:** C#, ASP.NET Core, Entity Framework Core, SQL Server, Azure, Umbraco, Unity

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Run tests:

```bash
npm run test
```

Run linting:

```bash
npm run lint
```

## Project Structure
